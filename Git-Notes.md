### Open GIT Bash
- Creating directory <br>
```
`mkdir (name)`
```


- Changing directory
```
`cd (name)`
```


- Intialising the repository on your system
```
`git init`
```

### Open VSC
- Open file -> Find directory in PC/Disk > Users
- Make any file (add extension `.py`, `.`)
- Open terminal, open drop down menu and select GIT Bash
